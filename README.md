# Proyecto de Inicio de Sesión

Este es un proyecto simple de una página de inicio de sesión.

## Descripción

Este proyecto incluye una página de inicio de sesión con campos para el correo electrónico y la contraseña. Está construido con HTML, CSS y JavaScript.

## Estructura del Proyecto

- `index.html`: La página principal del proyecto.
- `styles.css`: Los estilos CSS para la página.
- `script.js`: El archivo JavaScript para la funcionalidad de la página.

## Cómo Usar

1. Clona este repositorio en tu máquina local:
    ```bash
    git clone https://github.com/abdeltorres05/Login.git
    ```

2. Navega a la carpeta del proyecto:
    ```bash
    cd Login
    ```

3. Abre el archivo [index.html](http://_vscodecontentref_/0) en tu navegador web preferido.

## Funcionalidades

- Validación de campos de correo electrónico y contraseña.
- Mensajes de error si los campos están vacíos o no cumplen con los requisitos.
